Данное приложение реализует построение полигона из точек по входным данным в виде адресса субъекта и частоте точек (frequency),
которую можно задать через ComboBox.
Провайдером для визуализации полигона на карте могут выступать различные сервесы (Google, Yandex, Open Street Map, Bing и т.д.),
для этого необходимо заменить значение для GMapProviders в коде.
В каждой интерации поиска создается файл в формате json по указанному пути (@"D:\map.json").
